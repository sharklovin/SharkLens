Publish Checklist
Use this before making the repository public.

1. Clean the repository
Remove .env files
Remove secrets, tokens, and credentials
Remove private PCAPs and any non public evidence
Replace internal file paths with safe examples
Rename anything messy or unclear
2. Polish the repo
Add screenshots to docs/screenshots/
Add one sanitized example input
Add one sanitized example output
Review README for clear wording
Confirm license choice
3. Final Git steps
git init -b main
git add .
git commit -m "Initial public release"
git remote add origin https://github.com/YOUR_USERNAME/sharklens.git
git push -u origin main
4. GitHub profile steps
Pin SharkLens on your GitHub profile
Add a strong repo description
Add website or demo link if available
Upload a social preview image for better sharing
